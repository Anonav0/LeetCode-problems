# 🔹 LC 784 — Letter Case Permutation

### 🧠 Problem

Given a string `s`:

- Letters → can be **lowercase or uppercase**
- Digits → must stay same

👉 Return all possible permutations

---

### Example

```text
s = "a1b2"

Output:
["a1b2","a1B2","A1b2","A1B2"]
```

---

# 🔴 Step 1: Think Brute Force (Tree)

At each character:

👉 If it's a letter:

- choose lowercase
- choose uppercase

👉 If it's a digit:

- only one choice

---

# 🔥 This forms a **decision tree**

For `"a1b"`:

```
        ""
      /    \
     a      A
      \      \
      1      1
     / \    / \
    b   B  b   B
```

---

# 🔹 Step 2: Identify Pattern

👉 This is:

> **Generate all combinations → Backtracking**

---

# 🔹 Step 3: Core Idea

We build string step by step:

```text
index → which character we are processing
path → current string built so far
```

---

# 🔹 Step 4: Recursive Logic

At index `i`:

### Case 1: Digit

```text
only one choice → move forward
```

### Case 2: Letter

```text
branch into:
- lowercase
- uppercase
```

---

# 🔹 Step 5: Base Case

```text
if index == s.length → add result
```

---

# ✅ Java Code

```java
class Solution {
    public List<String> letterCasePermutation(String s) {
        List<String> result = new ArrayList<>();
        backtrack(s, 0, new StringBuilder(), result);
        return result;
    }

    private void backtrack(String s, int index, StringBuilder path, List<String> result) {
        if (index == s.length()) {
            result.add(path.toString());
            return;
        }

        char ch = s.charAt(index);

        if (Character.isDigit(ch)) {
            path.append(ch);
            backtrack(s, index + 1, path, result);
            path.deleteCharAt(path.length() - 1);
        } else {
            // lowercase
            path.append(Character.toLowerCase(ch));
            backtrack(s, index + 1, path, result);
            path.deleteCharAt(path.length() - 1);

            // uppercase
            path.append(Character.toUpperCase(ch));
            backtrack(s, index + 1, path, result);
            path.deleteCharAt(path.length() - 1);
        }
    }
}
```

---

# 🔥 Important Backtracking Pattern

👉 **Choose → Explore → Undo**

```text
1. Add choice
2. Recurse
3. Remove choice (backtrack)
```

---

# ⚡ Complexity

- Time: `O(2^n)` (for letters)
- Space: `O(n)` recursion
