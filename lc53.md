## 📜 Problem Statement

Given an integer array nums, find the with the largest sum, and return its sum.

Example 1:

Input: nums = [-2,1,-3,4,-1,2,1,-5,4]
Output: 6
Explanation: The subarray [4,-1,2,1] has the largest sum 6.

Example 2:

Input: nums = [1]
Output: 1
Explanation: The subarray [1] has the largest sum 1.

Example 3:

Input: nums = [5,4,-1,7,8]
Output: 23
Explanation: The subarray [5,4,-1,7,8] has the largest sum 23.

Constraints:

    1 <= nums.length <= 105
    -104 <= nums[i] <= 104

## Follow up: If you have figured out the O(n) solution, try coding another solution using the divide and conquer approach, which is more subtle.

## 🔍 Key Observations

- Subarray mentioned, max, negative -> Kadane's algo needed.

- at each i either that number or the prev sum will be the best subarray.

---

## 💡 Approach

### Approach 1: Kadane's algo + DP

🔹 Step 1: Brute Force Thinking

Try all subarrays:

Start index i
End index j

👉 Time = O(n^2) (or worse)

We can do better.

🔹 Step 2: Key Observation (Game Changer)

At any index i, ask:

👉 “Should I continue the previous subarray or start fresh?”

🔹 Step 3: Define State
dp[i] = maximum subarray sum ending at index i

⚠️ Important:
"ending at i" — this constraint is everything.

🔹 Step 4: Recurrence
dp[i] = max(nums[i], dp[i-1] + nums[i])

👉 Either:

Start new subarray from i
Extend previous subarray

```java
class Solution {
    public int maxSubArray(int[] nums) {
        int current = nums[0];
        int maxSum = nums[0];

        for (int i = 1; i < nums.length; i++) {
            current = Math.max(nums[i], current + nums[i]);
            maxSum = Math.max(maxSum, current);
        }

        return maxSum;
    }
}

class Solution {
    public int maxSubArray(int[] nums) {
        int n = nums.length;
        int max = nums[0];
        int curSum = 0;
        for(int i=0; i<n ; i++){
            if (curSum < 0 ) { // negative number hurt the sum so reset it.
                curSum = 0;
            }
            curSum += nums[i];
            max = Math.max(max, curSum);
        }
        return max;
    }
}
```
