## 📜 Problem Statement

You are given an integer array coins representing coins of different denominations and an integer amount representing a total amount of money.

Return the fewest number of coins that you need to make up that amount. If that amount of money cannot be made up by any combination of the coins, return -1.

You may assume that you have an infinite number of each kind of coin.

Example 1:

Input: coins = [1,2,5], amount = 11
Output: 3
Explanation: 11 = 5 + 5 + 1

Example 2:

Input: coins = [2], amount = 3
Output: -1

Example 3:

Input: coins = [1], amount = 0
Output: 0

Constraints:

    1 <= coins.length <= 12
    1 <= coins[i] <= 231 - 1
    0 <= amount <= 104

## 💡 Approach

### Approach 1: DP

🔹 Step 1: Brute Force Thinking

First instinct:

Try all combinations → recursion

For each coin:

take it → reduce amount

repeat

f(amount) = min(1 + f(amount - coin))

🚨 Problem:

Overlapping subproblems

Exponential time

🔹 Step 2: Identify DP Pattern

This is:

Unbounded Knapsack (minimization)

Why?

You can use a coin infinite times

You want minimum count

🔹 Step 3: Define State

👉 Let’s define:

    dp[i] = minimum coins needed to make amount i

🔹 Step 4: Recurrence

For each amount i:

    dp[i] = min(dp[i - coin] + 1)

for all coins where i - coin >= 0

🔹 Step 5: Base Case
dp[0] = 0

👉 0 coins needed to make amount 0

🔹 Step 6: Initialization

We need a "worst case":

    Arrays.fill(dp, amount + 1);

Why amount + 1?

    Max coins possible = amount (all 1s)

So this acts like infinity

🔹 Step 7: Build Bottom-Up

Now loop:

    for (int i = 1; i <= amount; i++) {
        for (int coin : coins) {
            if (i - coin >= 0) {
                dp[i] = Math.min(dp[i], dp[i - coin] + 1);
            }
        }
    }

🔹 Step 8: Final Answer

    return dp[amount] == amount + 1 ? -1 : dp[amount];

**Time Complexity:** O(amount \* coin)  
**Space Complexity:** O(amount)

```java
class Solution {
    public int coinChange(int[] coins, int amount) {
        int dp[] = new int[amount+1];
        Arrays.fill(dp, amount+1);
        dp[0]=0;

        for(int i=1; i < amount+1; i++){

            for(int coin : coins){

                if(i-coin >= 0){
                    dp[i] = Math.min(dp[i], 1+dp[i-coin]);
                }
            }
        }
        return dp[amount] == amount+1 ? -1: dp[amount];

    }
}
```
