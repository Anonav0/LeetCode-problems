## 📜 Problem Statement

Given a non-empty array of integers nums, every element appears twice except for one. Find that single one.

You must implement a solution with a linear runtime complexity and use only constant extra space.

Example 1:

Input: nums = [2,2,1]

Output: 1

Example 2:

Input: nums = [4,1,2,1,2]

Output: 4

Example 3:

Input: nums = [1]

Output: 1

Constraints:

    1 <= nums.length <= 3 * 104
    -3 * 104 <= nums[i] <= 3 * 104
    Each element in the array appears twice except for one element which appears only once.

## 🔍 Key Observations

- This can be solved with hashsets and brute force also.

## 💡 Approach

### Approach 1: Bit Manupilation (XOR ops)

- doing xor opertions with the prev elements in the array.
- it will give the not duplicate number. (seems like magic but it's maths.)
  **Steps**

1. Loop through the array
2. do xor ^= arr[i];
3. return xor;

**Time Complexity:** O(n)  
**Space Complexity:** O(1)

```java
# code hereclass Solution {
    public int singleNumber(int[] nums) {
        int xor =0;
        for(int n : nums){
            xor ^= n;
        }
        return xor;
    }
}
```
