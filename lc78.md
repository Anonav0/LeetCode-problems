# 🔹 LC 78 — Subsets

### 🧠 Problem

Given an array `nums[]`, return **all possible subsets** (power set).

Example:

```text
nums = [1,2]

Output:
[[], [1], [2], [1,2]]
```

---

# 🔴 Step 1: Think Like Choices

For every element, you have **2 choices**:

```text
Include it OR Exclude it
```

---

# 🔥 This creates a binary decision tree

For `[1,2,3]`:

```text
            []
        /        \
      [1]        []
     /   \      /   \
 [1,2] [1]   [2]   []
```

👉 Total subsets = `2^n`

---

# 🔹 Step 2: Core Pattern

👉 This is:

> **Include / Exclude pattern**

---

# 🔹 Step 3: Backtracking Template

We track:

- `index` → current position
- `path` → current subset

---

# 🔹 Step 4: Important Insight

👉 At **every step**, the current `path` is a valid subset

So we **add it immediately**

---

# ✅ Java Code

```java
class Solution {
    public List<List<Integer>> subsets(int[] nums) {
        List<List<Integer>> result = new ArrayList<>();
        backtrack(nums, 0, new ArrayList<>(), result);
        return result;
    }

    private void backtrack(int[] nums, int index, List<Integer> path, List<List<Integer>> result) {
        result.add(new ArrayList<>(path)); // add current subset

        for (int i = index; i < nums.length; i++) {
            path.add(nums[i]);                // choose
            backtrack(nums, i + 1, path, result); // explore
            path.remove(path.size() - 1);     // undo
        }
    }
}
```

---

# 🔥 Why This Works (Very Important)

Instead of binary recursion:

👉 We use a **for-loop to control choices**

This avoids duplicate work and keeps subsets ordered.

---

# 🔹 Alternative Way (Pure Include/Exclude)

```java
private void dfs(int[] nums, int i, List<Integer> path) {
    if (i == nums.length) {
        result.add(new ArrayList<>(path));
        return;
    }

    // include
    path.add(nums[i]);
    dfs(nums, i + 1, path);
    path.remove(path.size() - 1);

    // exclude
    dfs(nums, i + 1, path);
}
```

---

# ⚡ Complexity

- Time: `O(n * 2^n)`
- Space: `O(n)` recursion

---

# 🔥 Pattern Summary

| Problem      | Pattern         |
| ------------ | --------------- |
| Subsets      | include/exclude |
| Letter Case  | branching       |
| Permutations | swapping        |

---

# 🧠 Deep Insight (Don’t Miss)

👉 In subsets:

- Order does NOT matter
- So we use `index` to avoid duplicates
